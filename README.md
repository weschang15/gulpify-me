# Gulpify Me

An organized gulp starter kit that enables automating of frontend development tasks such as:

1. BrowserSync 
2. CSS Autoprefixing
3. Directory/File Deletion
4. CSS Minification
5. File Concatenation
6. HTML Minification
7. Image Optimization
8. File Renaming
9. Stylus Compiling
10. JS Minification
11. Critical CSS Generation
